# 为什么要用 lua

# lua

Lua 是一个小巧的脚本语言。作者是巴西人。该语言的设计目的是为了嵌入应用程序中，从而为应用程序提供灵活的扩展和定制功能。一个完整的 Lua 解释器不过 200k，在目前所有脚本引擎中，Lua 的速度是最快的。这一切都决定了 Lua 是作为嵌入式脚本的最佳选择。

# lua 的应用场景

游戏开发
独立应用脚本
Web 应用脚本
扩展和数据库插件如：MySQL Proxy 和 MySQL WorkBench
安全系统，如入侵检测系统

# lua 作图

Lua 制图是指通过使用 YDWE 的 Lua 引擎来使用 Lua 语言进行魔兽争霸 3 地图制作的技术。

引用大佬的话：

actboy168: 用过的人都再也不想有 J 做图了。

感谢 actboy168 的辛苦付出。。。

# 梦开始的敌方

Lua 作为目前最为流行的、免费轻量级嵌入式脚本语言，在很多工业级的应用程序中被广泛应用，如 Adobe's Photoshop，一些游戏引擎也是使用 lua 作为脚本语言，如 Unity。通过制作魔兽地图来学习 lua 是一个非常棒的途径。未来使用其他的游戏引擎也能够快速入手。
中二少年谁还没个游戏梦。

# 准备：搭建环境

去微软官方下载 vscode 软件下载地址.
安装 Lua 插件，在 vscode 中按 Ctrl+P，输入 lua 下载星最多下载量多大的。
下载最新版的 Ydwe。
下载最新版的内置 Japi。(也可以跳过此步，YDWE 的 JAPI 只能在 11 对战平台运行)。
